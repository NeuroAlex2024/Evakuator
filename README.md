Современный, конверсионный сайт для службы эвакуации автомобилей в Москве и Московской области.

## 🎯 Особенности сайта

### Маркетинговые фишки
- ✅ **Hero-секция** с мощным УТП "Эвакуатор за 30 минут"
- ✅ **Онлайн калькулятор** стоимости с мгновенным расчетом
- ✅ **Модальное окно** обратного звонка с таймером
- ✅ **Социальные доказательства** - отзывы и цифры

### Технические особенности

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

### 2. Установка браузеров для тестирования
```bash
npm run install-browsers
```

### 3. Запуск локального сервера
```bash
npm run serve
```
Сайт будет доступен по адресу: http://localhost:3000

### 4. Запуск тестов
```bash
# Запуск всех тестов
npm test

# Запуск тестов с видимым браузером
npm run test:headed

# Интерактивный UI для тестов
npm run test:ui

# Отладка тестов
npm run test:debug

# Просмотр отчета о тестах
npm run test:report
```

### Частые проблемы
1. **Тесты не запускаются**: Установите браузеры `npm run install-browsers`
2. **Сайт не открывается**: Проверьте порт 3000, используйте другой
3. **Анимации тормозят**: Отключите в CSS `animation: none`
